# HWH Extensions GreasyFork Artifacts

Публичное зеркало содержит только собранные `.user.js` и `.meta.js` для скриптов с включенной публикацией.
Исходники живут в приватном репозитории, потому что публичная витрина не обязана показывать всю кухню.

GreasyFork должен синхронизироваться с raw URL файлов из этой ветки.
